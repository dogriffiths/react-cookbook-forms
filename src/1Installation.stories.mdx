# Installation

Install the library with `npm install react-cookbook-forms` or `yarn add react-cookbook-forms`. If you are
going to use the built-in field components (`Text`, `Select`, `TextArea`) then you should also import the
styles:

    import {Form, Field, FieldGroup, Text, Select} from "react-cookbook-forms";
    import 'react-cookbook-forms/dist/index.css';
    export default () => {
        const [value, setValue] = useState()
        return <Form onChange={setValue}>
            <Field name='yourName'>
                <Text/>
            </Field>
            <Field name='startDate'>
                <Text type='date'/>
            </Field>
            Current value: {JSON.stringify(value)}
        </Form>
    }
